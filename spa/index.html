Front-end development

This is a collection of links cataloguing front-end development best practices and conventions. It is far too short to cover the entire gamut of front-end subjects and I'm sure I've missed someone who should most definitely be on here.

  • Ponyfoo is one of the best JavaScript resources out there.
  • Mozilla Developer Network is a great reference work for everything and anything web, including but not limited too, HTML, CSS, JS, SVG and more.
  • David Walsh often has helpful tips about HTML API's and new JavaScript functionality.
  • Addy Osmani is a great front-end developer working at Google. He has published Essential JavaScript Design Patterns, which is available for free online.
  • Lea Verou is most certainly the queen of CSS. She has written the great book CSS Secrets and contributes directly to the CSS Working Group, defining the next versions of CSS Modules.
  • Where Lea is CSS' queen Sara Soueidan rules the lands of SVG.
  • CSS Tricks is a good resource for anything CSS.
  • Heydon Pickering's website is a great place to look for accessibility advice.
